Сканер TCP и UDP портов.
Программа проверяет открыты ли порты в заданном диапазоне,
определяет протокол, который работает на открытом порте.
Поддерживаемые протоколы:  http, smtp, pop3, dns, sntp

Примеры запуска:
python3 scanner.py -h
python3 scanner.py 127.0.0.1 53 123

Для подробной информации об аргументах программы выполните:
python3 scanner.py -h
